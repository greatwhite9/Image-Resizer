# Image-Resizer
It is a versatile Python application designed to simplify the process of resizing images. Whether you need to scale down high-resolution photos or adjust the dimensions of images for various purposes, this tool provides an easy-to-use graphical interface for the job.

## Features

- Load an image file for resizing.
- Select the output folder where the resized image will be saved.
- Specify the new width and height for the resized image.
- Supports various image formats.
- User-friendly GUI.

## Prerequisites

Before you use this tool, you need to have the following installed:

- Python
- OpenCV (cv2)
- Tkinter
